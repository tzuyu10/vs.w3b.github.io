Visual Studio Basic Environment

Menu Bar

- Contains Commands for managing IDE (Integrated Development Environment) and for developing, maintaining, and executing programs.

- Menus contain groups of related commands which performs specific sections.

Toolbar

- Contains buttons that provide quick access to commonly used commands.

- Some buttons/icons contain a drop-down arrow which displays additional commands.

Properties Window

- Window that displays the properties for a form or control.

- Properties specify the characteristics such as size, position, name and text of a form or object

Toolbox

- It contains controls that you use to add objects like labels, graphic images, buttons, list boxes, text box, etc.

Solution Explorer

- Window that gives you a tree-structured view of all the window files in the application

- Displays the project name, forms, modules, classes, and more.

Form Designer

- Default Name is Form1.vb.

- Located at the center of the Window Screen.

- It is where you design the user interface of your application.

- Where you see and interact with when running the program.

Status Bar

- Display Information about the current state of the elements of the IDE window.
