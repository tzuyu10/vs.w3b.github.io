How to Setup Visual Studio

In order to setup Visual Studio, you need to download and install it from the website of Microsoft.

After the installation, you need to select .Net desktop development in order to make a Windows form application.

Next is to open Visual Studio and create a new project

In creating a new project, the project template should be Windows Forms App with Visual Basic as the programming language.

Lastly, you can rename the file and change its save file location

Now you can create the file and see the Visual Studio Environment.
