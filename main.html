Visual Studio (VS)

Visual Studio is a software development system developed by Microsoft in 1997.
Its main function is to provide an environment that makes program development easier and faster.

It is an Integrated Development Environment (IDE) that can be used to develop
console and Graphical User Interface (GUI) applications.

Visual Basic (VB)

Is a computer programming languange developed by Microsoft in 1991.
It is an object-oriented programming language

It is used to develop programs that can be used as front end application to a database system.
