About

This website can help students to learn about the basics of using Visual Studio and has a basic introductory into coding using Visual Basic.

It teaches students on how to use and navigate Visual Studio

It also teaches students about syntax and semantics of Visual Basic

This website is for educational purposes.

This is an output project for Practical Research 2 made by Grade 12 Programming students of Flora A. Ylagan High School.
